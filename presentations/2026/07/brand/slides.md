# Brand Styleguide

Aman Sanoj

---

## Brand System

Colors and typography, all in one place.

* Interactive styleguide with Dark and Light mode support 
* NPM package availability for seamless integration 
* Direct links to preview themes in RealTimeColors 

---

## Colors

Comprehensive token system for the brand palette.

* **Categories:** Primary, Secondary, Accent, Text, and Background 
* **Structured Shades:** Base colors accompanied by scaled numeric shades 
* **Quick Copy:** Interactive badges to instantly copy HEX, RGB, or OKLCH values 

---

## Typography

The three typefaces that shape the brand's voice.

* **Display:** Used for large, impactful headings 
* **Body:** Optimized for legibility and long-form reading 
* **Mono:** Distinct styling for code blocks and terminal output 
* **Integration:** Direct links to explore specimens and weights on Google Fonts 

---

## Core Technologies

* Basecoat & Tailwind CSS 
* Google Fonts 
* Phosphor Icons & Simple Icons
